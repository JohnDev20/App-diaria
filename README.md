# App Diaria

## Overview
App Diaria is a mobile application built with Natively, designed to help users manage their daily activities efficiently. The app focuses on simplicity and user-friendliness, providing a seamless experience to keep track of tasks, reminders, and schedules.

## Features
- **Task Management**: Easily create, edit, and delete tasks.
- **Reminders**: Set reminders for important tasks.
- **Calendar Integration**: Sync with your calendar to ensure you never miss an important event.
- **User Profiles**: Allow users to create profiles and save their settings.
- **Notifications**: Get notified for upcoming tasks and reminders.

## Getting Started
To get started with the App Diaria, follow the instructions below:

### Prerequisites
- **Node.js**: Make sure you have Node.js installed on your machine.
- **Natively**: You will also need the Natively SDK installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JohnDev20/App-diaria.git
   cd App-diaria
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
Once the application is running, you can:
1. Create a new task by navigating to the 'Tasks' section.
2. Set reminders by clicking on the task and choosing a reminder option.
3. Sync your calendar under settings to see all upcoming events.
